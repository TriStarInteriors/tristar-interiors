# Job Components Form

## Purpose
Tracks each operational section inside a job.

Examples:
- Cabinets
- Countertops
- Closets
- Flooring

## Fields

| Field Name | Type |
|---|---|
| Job | Lookup -> Jobs |
| Component_Type | Dropdown |
| SQFT | Number |
| Waste_Percent | Number |
| Carton_Coverage | Number |
| Cartons_Needed | Number |
| Cost_Per_Carton | Currency |
| Material_Cost | Currency |
| Labor_Rate | Currency |
| Labor_Cost | Currency |
| Trim_Cost | Currency |
| Underlayment_Cost | Currency |
| Demo_Cost | Currency |
| Total_COGS | Formula |
| Installed | Checkbox |

## Component Type Options

- Cabinets
- Countertops
- Closets
- Flooring
